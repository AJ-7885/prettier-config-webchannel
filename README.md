# prettier-config-webchannel

> Sharing a Prettier configuration [Prettier](https://prettier.io) config.

## Usage

**Install**:

```bash
yarn add --dev prettier-config-webchannel
```

**Edit package.json**:

```jsonc
{
  // ...
  "prettier": "prettier-config-webchannel"
}
```
